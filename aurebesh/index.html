aurebesh.js

aurebesh.js - 将JavaScript翻译成其他书写系统。

在下面的输入中键入最多九个符号的列表。 然后，您的符号将用于生成一小段有效的JavaScript，无需任何额外代码即可执行。

* 由 Martin Kleppe aka 制作@aemkei.
* 更多有趣的代码在 aem1k.com.

字符表：

↑ 编辑上面的 "字符表：" 。

示例代码是 `alert(1)` - 但其他代码也可以。 一般来说，字符数量不会超过140个。 你可以分享你的 URL 。

Examples

这里有一些真实和想象的书写系统的例子。 Aurebesh 基于《星球大战》星系中最常用的字母。

如何工作的：

这是ABC的格式化版本。仔细阅读，你会明白的。

A = ''              // 空字符串
B = !A + A          // "true"
C = !B + A          // "false"
D = A + {}          // "[object Object]"
E = B[A++]          // "t" = "true"[0]
F = B[G = A]        // "r" = "true"[1]
H = ++G + A         // 2, 3
I = D[G + H]        // "c"

B[
  I +=              // "c"
    D[A] +          // "o" = "object"[0]
    (B.C+D)[A] +    // "n" = "undefined"[1]
    C[H] +          // "s" = "false"[3]
    E +             // "t"
    F +             // "r"
    B[G] +          // "u" = "true"[2]
    I +             // "c" = "[object]"[5]
    E +             // "t"
    D[A] +          // "o" = "[object]"[1]
    F               // "r"
][
  I                 // "constructor"
](
  C[A] +            //  "a"
  C[G] +            //  "l"
  B[H] +            //  "e"
  F +               //  "r"
  E +               //  "t"
  "(A)"             // "(1)"
)()
